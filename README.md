# Social Graph Project

A thread-safe, in-memory social network graph implemented in Python. This project supports user management, friendship management, mutual friends, shortest path, friend recommendations, and connected components, with efficient algorithms and comprehensive test cases.

## Features
- Add and remove users
- Add and remove friendships
- Find mutual friends
- Find shortest path between users (degrees of separation)
- Recommend friends (with optional top-k)
- Find all connected components in the network
- Thread-safe operations using Python threading
